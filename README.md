# 서비스 소개
* 서비스명: 모두의 AI, MoAI
* 서비스설명: <br>
채팅, 일정 및 프로젝트 진행도 공유, 문서 승인 등 기본적인 업무 보조 도구 <br>
ollama를 활용한 Local LLM API를 포함한 Express 기반 서버 <br>
채팅, 일정 등 업무 보조 도구의 내용, 사용자가 직접 입력한 문서 파일의 내용을 기반으로 대답해주는 RAG Model <br>
